Instructions for using Github

1. Đăng ký tài khoản Github tại trang chủ: https://github.com




2. Sau khi đã có tài khoản, bạn đăng nhập vào Github và có thể tạo một Repository, Repository này yêu cầu tên

giống như tài khoản của bạn và có đuôi github.io







...và Repository đã được tạo ra, và bạn có thể đăng các file html, css... của mình lên.




Nếu muốn tạo trang chủ, bạn chỉ cần tạo ra một trang index.html, vậy là bạn đã có trang web cơ bản của bạn.
